# Java 字节码规范

## 1. 字节码常见指令
| 操作码 | 助记符 | 释义 |
| ---| --- | --- |
| 00 | nop | 什么都不做|
| 01 | aconst_null | 将 `null` 推送到栈顶 |
| 02 | iconst_m1 | 将整数 `-1` 推送至栈顶 |
| 03 | iconst_0 | 将整数 `0` 推送至栈顶 |
| 04 | iconst_1 | 将整数 `1` 推送至栈顶 |
| 05 | iconst_2 | 将整数 `2` 推送至栈顶 |
| 06 | iconst_3 | 将整数 `3` 推送至栈顶 |
| 07 | iconst_4 | 将整数 `4` 推送至栈顶 |
| 08 | iconst_5 | 将整数 `5` 推送至栈顶 |
